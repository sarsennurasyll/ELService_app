# ELService

Мобильное приложение для маркетплейса услуг по ремонту бытовой техники.

## Стек

Flutter 3+, Dart, Material 3, Riverpod, GoRouter, Dio, Freezed и Json Serializable.

## Структура проекта

Проект организован по Feature First и Clean Architecture.

- `lib/app` — точка сборки приложения, маршрутизация, тема и запуск.
- `lib/core` — общая техническая инфраструктура.
- `lib/shared` — переиспользуемые элементы, общие для нескольких функций.
- `lib/features` — независимые функциональные модули приложения.
- `lib/l10n` — файлы локализации.

На текущем этапе создана только структура каталогов. Код, зависимости и интерфейс будут добавляться отдельными задачами.
